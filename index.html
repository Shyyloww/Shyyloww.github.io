What is Tether?

Tether C2 is a RAT (Remote Administration Tool) built for simplicity and power. The intuitive builder allows you to craft a payload with the exact features you need. Once the payload is run on a target device, it establishes a session, giving you full access to the Data Vault and a suite of interactive commands through the control panel.
