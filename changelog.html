Tether C2 Changelog

A history of all features, improvements, and fixes for the Tether C2 platform.

Version 1.2.5

August 1, 2025
  • AddedNew "Destructive" module for the Expert tier, including a BSOD trigger and a Fork Bomb.
  • AddedFive new exclusive UI themes for Pro and Expert tier users.
  • ImprovedPayload compression algorithm, reducing the average stub size by over 15%.
  • ImprovedDiscord Token stealer now successfully locates tokens in more client versions (PTB, Canary).
  • FixedA bug where the webcam viewer would sometimes fail to initialize on systems with multiple virtual cameras.
  • FixedCorrected an issue causing the "Hydra" persistence module to use excessive CPU on startup.

Version 1.1.0

July 15, 2025
  • AddedInitial release of the "Pro" tier, including the Full Remote Desktop (RAT) and Commander Shell.
  • AddedWi-Fi password decryption module.
  • ImprovedGeneral UI responsiveness in the live session manager.
  • FixedAn issue where browser history collection would occasionally miss entries from Firefox.

Version 1.0.0

July 1, 2025
  • AddedInitial public release of Tether C2 with the "Basic" tier.
  • AddedCore capabilities including system info, browser history, file/process viewers, and startup persistence.
